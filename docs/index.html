Quick Setup

Install the Salesforce MCP server in your preferred IDE. The server provides seamless integration with Salesforce orgs, enabling your AI agent to access and manipulate Salesforce data directly from your development environment.

{
  "mcpServers": {
    "salesforce": {
      "command": "npx",
      "args": ["@your-org/salesforce-mcp"]
    }
  }
}

Copy this configuration to your IDE's MCP settings and you're ready to go!
